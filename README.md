# Робо-рука

У робо-руки есть два состояния:

- `сжата`
- `разжата`

## Как запустить

Откройте терминал в папке проекта.

Введите команды:

```powershell
pip install -r requirements.txt
python app.py
```

Откройте сайт:

```text
http://127.0.0.1:5000
```

## Как управлять

Сжать руку:

```text
http://127.0.0.1:5000/on
```

Разжать руку:

```text
http://127.0.0.1:5000/off
```

## Где хранится статус

Статус хранится в файле:

```text
hand_status.txt
```

Это простая замена базы данных.

В файле написано одно слово:

- `сжата`
- `разжата`

## Примечание

В репозитории также лежит описание робо-руки в формате PDF.

## Как остановить

В терминале нажмите:

```text
Ctrl+C
```
